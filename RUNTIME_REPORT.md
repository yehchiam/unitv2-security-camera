## Runtime Log

Performance data is logged every 5 minutes to `/mnt/sdcard/perf_log.csv` on the UnitV2.

### CSV Format
```
timestamp,uptime_sec,state,cpu_pct,mem_total_mb,mem_used_mb,mem_avail_mb,clips,storage_mb,interval_sec
```

- **state**: I=Idle, M=Motion detected, R=Recording
- **cpu_pct**: CPU usage percentage (sampled over 5-min interval)
- **clips**: Total number of recorded clips on SD card
- **storage_mb**: Total storage used by clips in MB
